Progressive Web Apps

Created by Fábio Antunes

A progressive web app provides an app-like user experience that is low friction and is built using modern web capibilities

Unordered list with some definitions

  • PWAs are the future!
  • PWAs aren't packaged, distributed or deployed through app stores
  • PWAs tend to behave, look and feel like Native Apps
  • PWAs request permissions in an incremental way
  • PWAs start as a website and become progressively more “app”

PWAs key features

  • Performance!
  • Background Sync
  • Durable storage
  • Web Bluetooth, WebUSb
  • Web NFC
  • Improved WebRTC
  • Offline
  • Push Notifications!!
  • Store and cache the shell of your website
  • Upload photos or files in the background
  • Configure and control devices
  • Transfer data between devices
  • More Control over camera
  • Last but not least, Push notifications
